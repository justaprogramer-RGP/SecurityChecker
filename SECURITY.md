# Security Policy for SecurityChecker

Thank you for using SecurityChecker! Your security reports help us keep this project safe and reliable.

## Reporting a Vulnerability

If you discover a security vulnerability, please follow these steps:

1. **Do NOT open a public GitHub issue**.  
2. Send an email to: **jack.justapro@gmail.com**  
   - Include detailed information about the vulnerability.  
   - Steps to reproduce it, if possible.  
3. We will respond **as soon as possible**.

## Security Updates

- We will release fixes promptly after verifying reported vulnerabilities.  
- All security updates will be documented in the repository releases.  

## Additional Notes

- Thank you for helping keep SecurityChecker safe.  
- Do not exploit reported vulnerabilities for malicious purposes.
